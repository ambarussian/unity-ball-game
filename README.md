# ⚽ Ball Game
This Unity project is a simple game where the player controls a ball within a bounded area with the objective of collecting squares scattered around the scene.
<br><br>

## How it works
- The player controls a **ball** that can move freely within a limited area.
- Inside this space, there are several rotating **squares** randomly positioned.
- The objective is to **collect all squares**.
- Each collected square increases the player's score on the scoreboard.
- The game ends when the player collects all 5 squares.
<br><br>

## 🎮 Controls

| Key | Action            |
|-----|-------------------|
| W / ↑ | Move up         |
| S / ↓ | Move down       |
| A / ← | Move left       |
| D / → | Move right      |

<br>

## Game mechanics

- The ball cannot leave the bounded area and is confined to the platform.
- Squares disappear when collected.
- It is possible to see how many squares have been collected on the scoreboard.
- The game can be restarted.
